## 🧠 How It Works

### Scroll Animation System

Elements are initially hidden and revealed when they enter the viewport.

```js
entry.target.classList.add('animate');
```

This creates a smooth **fade + slide-up effect** on scroll. 

---

### Skill Animation

Skill bars animate dynamically when the skills section becomes visible.

---

### Form Validation

Real-time validation for:

* Name
* Email
* Subject
* Message

Ensures better UX before submission.
